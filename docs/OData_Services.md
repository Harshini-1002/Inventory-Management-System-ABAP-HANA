# OData Services

## What is OData?

OData (Open Data Protocol) is used in SAP to expose business data so that applications like SAP Fiori can consume it.

It allows applications to perform Create, Read, Update, and Delete (CRUD) operations through REST APIs.

---

## Common Operations

### GET
Retrieve product details.

### POST
Create a new product.

### PUT
Update product information.

### DELETE
Remove a product.

---

## Why OData?

- Supports SAP Fiori applications
- Standard REST API protocol
- Easy integration with frontend applications
