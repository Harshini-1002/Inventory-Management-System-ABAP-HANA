# CDS Views

## What are CDS Views?

Core Data Services (CDS) Views are used in SAP to define database models at the application level.

They allow developers to retrieve and combine data efficiently while taking advantage of SAP HANA performance.

---

## Why use CDS Views?

- Better database performance
- Reusable data models
- Reduced application logic
- Supports OData service generation
- Optimized for SAP HANA
